## 3. 情境限制與體感警告 (特殊防呆)

這裡列出系統中目前偵測的「語意/情境衝突」，當玩家不小心穿錯時，會**立刻彈出**對應的警示文案：

| 觸發情境 (條件) | 系統警告標誌與評語文案 |
| :--- | :--- |
| **冬衣夏穿**<br>（在「熱」關卡穿著羽絨衫、膨線衫或圍巾） | 太陽好大！小主人汗流浹背！大夏天穿厚重的羽絨衫或膨線衫實在太悶熱了，快去幫模特兒換上舒適輕便的短衫吧！ |
| **夏衣冬穿**<br>（在「冷」關卡穿著短衫、短褲或裙） | 冷風吹來～小主人在瑟瑟發抖！你雖然寫對了客語單字，但是冬天穿短袖短褲會著涼喔！快幫模特兒換成防寒的羽絨衫或長褲吧！ |
| **藍衫重複染色**<br>（穿藍衫卻配上其他顏色詞） | 「藍衫」本身已具備藍色，不可再搭配其他顏色形容詞。 |
| **打籃球穿長褲**<br>（在籃球題穿著長褲） | 語意不協調：打籃球要求手腳靈巧好活動，搭配「長褲」可能限制劇烈跑跳。 |
| **非雨天/非打掃穿雨鞋**<br>（在一般晴天穿水靴筒） | 情境不協調：非下雨或打掃情境搭配雨鞋「水靴筒」不合語意。 |
| **非游泳穿泅水帽/衫**<br>（在日常場合穿戴泳裝） | 情境不協調：非水上活動情境搭配「泅水帽/泅水衫」不合語意。 |
| **涼爽天氣穿長褲或包鞋**<br>（在涼爽情境下穿長褲或包鞋） | 語意不協調：此題敘事句強調「涼爽」，搭配「長褲」或包鞋「鞋」體感溫度較高，較不符合涼爽感。 |
